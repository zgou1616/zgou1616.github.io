# Zheming Gou — Research Portfolio

Source files for [zgou1616.github.io](https://zgou1616.github.io/), a responsive academic and professional profile highlighting research interests, peer-reviewed publications, ongoing manuscripts, education, and selected research activities.

## Structure

- `index.html` — site content and navigation
- `assets/css/style.css` — responsive visual design
- `assets/images/profile.jpg` — profile photograph
- `assets/images/favicon.svg` — browser icon

The site is static and requires no build step. It can be served directly by GitHub Pages from the repository's `main` branch.

## Maintenance

Publication entries link to durable DOI or arXiv records. Update the ongoing-work status and the “Last updated” date when a manuscript changes status or new work is added.
